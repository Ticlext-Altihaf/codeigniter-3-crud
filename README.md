# English
## Prerequisites

1. Create database `crud`
2. Create table `users`
```sql
CREATE TABLE `users` (
id int(11) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
nama VARCHAR(255) NOT NULL,
alamat TEXT NOT NULL,
perkerjaan VARCHAR(255) NOT NULL
);
```
